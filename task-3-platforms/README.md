# Reflection Notes on Coding Practice and Collaboration Tools

This task helped me understand the importance of using the right digital tools for learning, practicing, and collaborating.

In Part A, I created an account on a coding platform and solved a beginner-level problem. This gave me hands-on experience in coding and improved my logical thinking and problem-solving skills. It also introduced me to how such platforms can be used regularly to strengthen programming knowledge.

In Part B, I created a Google Form as a Digital Literacy Awareness Quiz. While designing the form, I included both multiple-choice and short-answer questions, which helped me understand how to structure questions effectively. I also learned how responses are collected and organized automatically, making it useful for surveys and data analysis.

Overall, this task increased my confidence in using both coding and collaboration tools. It showed me how these platforms can support academic learning, teamwork, and skill development, which are essential for my future studies and career.
